## ğŸš€ Getting Started

### Prerequisites
```bash
# Node.js and npm
node --version
npm --version

# Python
python --version

# Git
git --version
```

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/saurabhvishwakarma419/Digital-Garden.git
cd Digital-Garden
```

2. **Navigate to specific project**
```bash
cd "Web Development/Task Management App"
# or
cd "Data Science & Machine Learning/Predictive Model"
```

3. **Install dependencies**
```bash
# For Node.js projects
npm install

# For Python projects
pip install -r requirements.txt
```

4. **Run the project**
```bash
# For web projects
npm start

# For Python projects
python main.py
```
