### General Setup

1. **Clone the repository**
```bash
git clone https://github.com/saurabhvishwakarma419/Digital-Garden.git
cd "Digital-Garden/Mobile App Development"
```

2. **Choose an app**
```bash
cd "Event Management System"  # or any other app
```

3. **Install dependencies**
```bash
# For React Native apps

cd ios && pod install && cd ..

# For Flutter apps
flutter pub get
```

4. **Run the app**
```bash
# React Native
npx react-native run-android  # Android
npx react-native run-ios       # iOS (macOS only)

# Flutter
flutter run
```
