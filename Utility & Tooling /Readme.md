## ğŸ¯ Common Use Cases

### AI Tutor
```bash
# Learn Python basics
python ai_tutor.py --topic "Python basics"

# Get help with code
python ai_tutor.py --code-review myfile.py

# Generate quiz
python ai_tutor.py --quiz "Data Structures" --questions 10
```

### Command Line Interface
```bash
# File operations
mycli files list --path ./
mycli files compress --input folder/ --output archive.zip

# Network utilities
mycli net ping google.com
mycli net speedtest
```

### File Organizer
```bash
# Organize Downloads folder
python organizer.py --path ~/Downloads --auto

# Preview organization (dry-run)
python organizer.py --path ~/Downloads --dry-run

# Watch folder for changes
python organizer.py --path ~/Downloads --watch
```

### Password Generator
```bash
# Generate secure password
python password_gen.py --length 20 --special

# Generate memorable passphrase
python password_gen.py --type passphrase --words 4

# Batch generation
python password_gen.py --count 10 --output passwords.txt
```

### Web Scraper
```bash
# Scrape website
python scraper.py --url "https://example.com" --output data.csv

# Use template
python scraper.py --template ecommerce --url "https://shop.com"

# Headless browser mode
python scraper.py --url "https://example.com" --browser --headless
```

## ğŸ› Troubleshooting

### Common Issues

**Issue: Module not found**
```bash
# Solution: Install dependencies
pip install -r requirements.txt
```

**Issue: API key error (AI Tutor)**
```bash
# Solution: Set environment variable
export OPENAI_API_KEY="your-key-here"
# Or create .env file
echo "OPENAI_API_KEY=your-key" > .env
```

**Issue: Permission denied**
```bash
# Solution: Add execute permission
chmod +x script.py
```

**Issue: Web scraper blocked**
```bash
# Solution: Use proxy or increase delay
python scraper.py --proxy "http://proxy:port" --delay 5
```
