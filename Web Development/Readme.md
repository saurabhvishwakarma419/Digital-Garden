
2. **Choose a project**

```bash

# For backend
cd ../server && npm install
```

4. **Configure environment variables**
```bash

cp .env.example .env
# Add your configuration
```

5. **Run the application**
```bash
# Development mode
npm run dev

# Production build
npm run build
npm start
```
