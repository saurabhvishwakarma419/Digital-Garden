### Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/saurabhvishwakarma419/Digital-Garden.git
cd "Digital-Garden/Game Development"
```

2. **Choose a game**
```bash

# or
```

```bash

# Using Node.js
npx serve
```

4. **Open in browser**
```
http://localhost:8000
```
